    pyenv的用法

    作者： xx 发布于： 2024-03-22 17:52:44 python
    python pyenv

    1. 显示可安装的python版本

    pyenv install --list

    2. 安装某个python版本

    pyenv install 3.12.1

    3. 显示本地已安装的Python版本

    pyenv versions

    4. 切换python版本

    pyenv global 3.12.1

    5. 显示Python执行文件路径

    pyenv which python3.12
