ubuntu16.04.4下安装mongodb的图形管理工具robomongo

mongodb mongo robomongo 图形管理工具
mongodb
发布日期:   2018-05-30
文章字数:   247

1. 官网下载

从官网 https://robomongo.org/download 下载当前最新的robomongo程序包robo3t-1.2.1-linux-x86_64-3e50a65.tar.gz

$ mkdir -p ~/soft
$ cd ~/soft/
$ wget -c "https://download.robomongo.org/1.2.1/linux/robo3t-1.2.1-linux-x86_64-3e50a65.tar.gz"

2. 解压并安装

$ cd ~/soft/
$ tar -xzvf robo3t-1.2.1-linux-x86_64-3e50a65.tar.gz
$ cd robo3t-1.2.1-linux-x86_64-3e50a65
$ cd bin/
$ ./robo3t
$ cd ~/soft/
$ sudo mv robo3t-1.2.1-linux-x86_64-3e50a65 /usr/local/robomongo

3. 添加环境变量使命令生效

添加环境变量ROBOMONGO_HOME， 并在环境变量PATH追加$ROBOMONGO_HOME/bin

$ sudo vim /etc/profile

追加的内容如下：

export ROBOMONGO_HOME=/usr/local/robomongo
export PATH=$PATH:$ROBOMONGO_HOME/bin

使文件/etc/profile内新添加的环境变量生效：

$ source /etc/profile

对于使用了zsh shell代替base shell的用户， 执行完source命令后会回到base shell下， 若继续使用zsh shell， 则需要logout再login， 命令robo3t才生效

$ robo3t --version

    distarch: x86_64
    target_arch: x86_64

OK, Enjoy it~
